# 🐔 La Granja - Juego Educativo de Preescolar

## 📝 Descripción

Juego educativo interactivo diseñado para niños de preescolar (3-5 años) que enseña a contar números del 1 al 10 de manera divertida y visual.

## 🎮 Juegos Incluidos

### 1. 🐔 Gallina - Canastas
- **Objetivo**: Identificar la canasta con el número correcto de manzanas
- **Habilidades**: Conteo visual, reconocimiento de números
- **Dificultad**: Configurable (2 o 3 opciones)

### 2. 🐴 Caballo - Saltar Vallas
- **Objetivo**: Saltar el número correcto de vallas
- **Habilidades**: Conteo secuencial, coordinación
- **Interacción**: Círculos animados tipo carrusel
- **Sonido**: Relincho del caballo al saltar

### 3. 🐮 Vaca - Pastitos
- **Objetivo**: Dar de comer el número correcto de pastitos a la vaca
- **Habilidades**: Conteo progresivo, motricidad
- **Interacción**: Click o arrastrar pastitos hacia la vaca
- **Sonido**: "Muuu" al comer cada pastito

## 🚀 Uso

1. Abrir `index.html` en un navegador moderno
2. Tocar la pantalla para activar sonidos (requerido por navegadores)
3. Seleccionar un juego
4. ¡Divertirse aprendiendo!
